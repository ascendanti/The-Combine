## Property Details

### Atlas Dispatch (Newsletter)

| Element | Specification |
|---------|---------------|
| **Frequency** | 3x weekly |
| **Format** | Curated + original analysis |
| **Topics** | MENA, geopolitics, tech-power |
| **Length** | 1,000-1,500 words |
| **Platform** | Substack/Ghost |

**Revenue Model:**
- Free tier: 3x week, abbreviated
- Paid tier ($10/mo): Full content, archive, community
- Target: 50K free → 2K paid = $240K ARR

### The Long View (Podcast)

| Element | Specification |
|---------|---------------|
| **Frequency** | Weekly |
| **Format** | Interview + analysis |
| **Length** | 45-60 minutes |
| **Guests** | Experts, practitioners, thinkers |

**Revenue Model:**
- Sponsorship: $50-100 CPM
- Target: 50K downloads/episode = $2.5K-5K/episode
- Premium episodes on Patreon

### Quick Context (Short Video)

| Element | Specification |
|---------|---------------|
| **Frequency** | Daily |
| **Format** | Explainer, reaction, analysis |
| **Length** | 60-180 seconds |
| **Platform** | TikTok, YouTube Shorts, Instagram Reels |

**Revenue Model:**
- Platform monetization (YouTube)
- Sponsorship integration
- Funnel to newsletter/podcast
